HOW TO USE IN POSTMAN
===============================================================================

1. Open Postman
2. Click "Import" button
3. Select "Raw Text" tab
4. Copy and paste any cURL command above
5. Click "Import"
6. The request will be created automatically!

OR

1. Import the collection: Rice_Mill_APIs.postman_collection.json
2. All requests will be ready to use!


===============================================================================
TROUBLESHOOTING
===============================================================================

Error: "Unauthorized (401)"
‚Üí Your token expired. Run LOGIN command again.

Error: "Batch number already exists (409)"
‚Üí Change the batchNumber to a unique value.

Error: "Connection refused"
‚Üí Application not running. Start it with: mvn spring-boot:run

Check if app is running:
‚Üí curl http://localhost:8080/actuator/health
